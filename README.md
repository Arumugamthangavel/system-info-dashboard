# 🖥️ System Info Dashboard

A simple Flask-based web application that displays real-time system information like CPU and memory usage. Then deployed it on docker.

---

## 🚀 Features
- Displays Operating System details  
- Shows CPU usage  
- Shows Memory usage  
- Lightweight and fast  

---

## 🛠️ Tech Stack
- Python (Flask)
- Docker
- Linux

---

## ⚙️ Run Locally

```bash
python app.py
